# rss-reader

## Tech

It is a Preact application that reads an RSS feed from https://beta.futurezone.de/rss.xml.

Build with [preact-cli](https://github.com/developit/preact-cli)

To clone this repo: 

`$ git clone https://github.com/Tina-Trillian/rss-reader

To start:

`$ cd rss-reader`  
`$ npm install`  
`$ npm run dev`
